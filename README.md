# Java POO - Exercícios Iniciais

## Sobre o projeto
Este projeto contém os primeiros exercícios práticos de Programação Orientada a Objetos em Java. Foi desenvolvido durante meus estudos na Alura.

## Exercícios realizados

### 1. Classe Pessoa
- Método `exibirMensagem()` - Exibe "Olá, mundo!" no console

### 2. Classe Calculadora
- Método `dobrarValor(int valor)` - Recebe um número e retorna o dobro

### 3. Classe Música
- Atributos: titulo, artista, anoDeLancamento, avaliacao, numDeAvaliacoes
- Métodos:
  - `exibirFichaTecnica()` - Mostra todas as informações da música
  - `avaliar(int nota)` - Adiciona uma nota à música
  - `calcularMediaAvaliacoes()` - Calcula a média das notas

### 4. Classe Carro
- Atributos: modelo, anoDocarro, cor
- Métodos:
  - `exibirFichaTecnica()` - Mostra modelo, ano e cor
  - `calcularIdadeDoCarro(int anoAtual)` - Calcula a idade do carro

### 5. Classe Aluno
- Atributos: nome, idade
- Método `exibirInformacoes()` - Mostra os dados do aluno

## Como executar

1. Clone o repositório
2. Abra no IntelliJ IDEA (ou qualquer IDE Java)
3. Execute a classe `Main.java`

## Tecnologias utilizadas

- Java 17
- IntelliJ IDEA

## O que aprendi

- Criar classes e objetos
- Declarar atributos (String, int, double)
- Criar métodos com e sem retorno (void, int, double)
- Usar parâmetros em métodos
- Entender a diferença entre `return` e `System.out.println()`
- Usar a classe Scanner para entrada de dados
